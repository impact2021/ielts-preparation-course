# Version 2.10 Testing Guide

**Version:** 2.10  
**Testing Date:** 2025-12-18

## Overview

This guide provides step-by-step instructions for testing all features and fixes introduced in version 2.10.

## Prerequisites

- WordPress site with IELTS Course Manager 2.10 installed
- At least one course with lessons
- At least one CBT (Computer-Based Test) exercise
- At least one standard quiz
- Test user account for taking quizzes

## Test Environment Setup

### 1. Create Test Quiz (CBT Layout)

**Admin Steps:**
1. Navigate to **IELTS Courses > Exercises**
2. Click **Add New**
3. Set title: "Test CBT Quiz v2.10"
4. Add quiz description
5. In Quiz Settings:
   - **Layout Type:** Computer-Based IELTS Layout
   - **Open as Popup/Fullscreen Modal:** ✅ Checked
   - **Scoring Type:** Percentage (Standard) or IELTS Band Score
   - **Timer (minutes):** 5 (optional, for timer testing)
6. Add Reading Text:
   - Title: "Passage 1"
   - Content: Add sample reading passage
7. Add Questions (mix of types):
   - **Question 1:** Multiple Choice (4 options)
   - **Question 2:** True/False/Not Given
   - **Question 3:** Fill in the Blank
   - **Question 4:** Multiple Choice (4 options)
   - **Question 5:** Fill in the Blank
8. Link questions to reading text if desired
9. Publish the quiz

### 2. Create Test Quiz (Standard Layout)

**Admin Steps:**
1. Create another quiz with **Layout Type:** Standard Quiz Layout
2. Add 3-5 questions (various types)
3. Publish the quiz

## Troubleshooting

### Issue: Score shows 0/0
**Solution:**
- Verify quiz has questions
- Check browser console for JavaScript errors
- Ensure user is logged in
- Clear browser cache and try again

### Issue: Visual feedback not showing
**Solution:**
- Hard refresh page (Ctrl+F5)
- Check if CSS loaded properly
- Verify no CSS conflicts with theme

### Issue: Modal not opening
**Solution:**
- Check browser console for errors
- Ensure "Open as Popup" is checked in quiz settings
- Try different browser
- Disable browser extensions

### Issue: Timer not working
**Solution:**
- Verify timer minutes set in quiz settings
- Check browser console for JavaScript errors
- Ensure quiz is in fullscreen mode
- Try refreshing page
