# XML Exercises

This directory contains XML exercise files for IELTS test preparation.

## Purpose

This folder stores:
- Generated XML exercise files
- Test imports ready for WordPress
- Exported test exercises

## File Format

All files in this directory should be WordPress XML export format (WXR) files that can be imported directly into the IELTS Course Manager plugin.

## Organization

XML files are named following the pattern:
- `[test-type]-test.xml` for demo tests
- `exercise-[test-name]-[date].xml` for practice exercises
