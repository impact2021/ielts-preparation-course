   - Type in the "Search Students:" box
   - Results filter automatically
   - No need to press Enter

## Troubleshooting

### Search not working?
- Ensure JavaScript is enabled in browser
- Check browser console for errors
- Try clearing browser cache
- Contact support if issue persists

### No results showing?
- Check spelling of search term
- Try shorter/partial search (e.g., "joh" instead of "johnson")
- Verify student exists under current tab (Active/Expired)
- Clear search to see if student appears in full list

### Search feels slow?
- This is unusual - search is client-side and instant
- Check browser performance
- Try with fewer browser tabs open
- Contact support with details
