# V2.22 Complete Testing Guide

**Version**: 2.22  
**Date**: December 19, 2024

## Overview

This guide provides comprehensive testing procedures for version 2.22, which includes:

**Previous fixes (already tested)**:
1. Question navigation margin in fullscreen mode
2. No-answer feedback visibility for multiple choice questions

**New enhancements (to be tested)**:
3. Multi-select answer highlighting after submission
4. Font size controls in timer bar
5. Progressive green navigation numbers for multi-select questions

## Prerequisites

- WordPress installation with IELTS Course Manager plugin v2.22
- At least one course created
- At least one quiz with multi-select questions
- User accounts for testing (admin and student)
- Multiple browsers for compatibility testing

## NEW FEATURES TESTING

### Feature 1: Multi-Select Answer Highlighting

#### Test 1.1: Fully Correct Multi-Select

**Setup:** Create a multi-select question with 3 correct answers (A, C, E)

**Steps:**
1. As student, select all 3 correct options (A, C, E)
2. Submit quiz

**Expected:**
- ✅ All selected options highlighted in solid green
- ✅ All navigation buttons green

**Screenshot Required:** Yes

---

#### Test 1.2: Partially Correct Multi-Select

**Steps:**
1. Select 2 correct (A, C) and 1 incorrect (B)
2. Submit quiz

**Expected:**
- ✅ A, C: solid green (correct selections)
- ✅ B: red (incorrect selection)
- ✅ E: light green border (missed correct answer)
- ✅ Nav buttons: 2 green, 1 red

**Screenshot Required:** Yes

---

#### Test 1.3: All Incorrect Selections

**Steps:**
1. Select only incorrect options (B, D)
2. Submit quiz

**Expected:**
- ✅ B, D: red (incorrect)
- ✅ A, C, E: light green border (all missed)
- ✅ All nav buttons red

**Screenshot Required:** Yes

---

### Feature 2: Font Size Controls

#### Test 2.1: Increase Font Size

**Steps:**
1. Open CBT quiz
2. Click "A+" button 4 times

**Expected:**
- ✅ Font increases from 16px to 24px (2px per click)
- ✅ Maximum of 24px enforced
- ✅ Only quiz content affected (not timer bar)

**Screenshot Required:** Yes (at 24px)

---

#### Test 2.2: Decrease Font Size

**Steps:**
1. Click "A-" button repeatedly

**Expected:**
- ✅ Font decreases to minimum 12px
- ✅ Stops at 12px (doesn't go smaller)

**Screenshot Required:** Yes (at 12px)

---

#### Test 2.3: Reset Font Size

**Steps:**
1. Set font to 22px
2. Click "A" (reset) button

**Expected:**
- ✅ Font returns to 16px immediately

---

#### Test 2.4: Font Size Persistence

**Steps:**
1. Set font to 20px
2. Refresh page
3. Navigate to different quiz

**Expected:**
- ✅ Font remains 20px after refresh
- ✅ Preference persists across quizzes

---

### Feature 3: Progressive Navigation Numbers

#### Test 3.1: During Quiz - Progressive Marking

**Setup:** Multi-select with 3 correct answers

**Steps:**
1. Check 1st checkbox → only 1st button green
2. Check 2nd checkbox → buttons 1-2 green
3. Check 3rd checkbox → buttons 1-3 green
4. Uncheck 1st → buttons 1-2 green (not 2-3)

**Expected:**
- ✅ First N buttons turn green where N = selections
- ✅ Unchecking updates correctly

**Screenshot Required:** Yes (showing progression)

---

#### Test 3.2: After Submission - Correct Count

**Setup:** Multi-select with 3 correct answers

**Scenario A:** 2 correct, 1 incorrect selected
**Expected:**
- ✅ 2 buttons green (correct selections count)
- ✅ 1 button red (missed answer)

**Scenario B:** 0 correct selected
**Expected:**
- ✅ All buttons red

**Screenshot Required:** Yes (both scenarios)
