üîß Technical Fix

The fix adds three key CSS properties to radio buttons and their text:

  • flex-shrink: 0 on the radio button prevents it from shrinking
  • align-self: flex-start keeps it at the top of the first line
  • flex: 1 and min-width: 0 on the text allows proper wrapping

These properties ensure the radio button always appears to the left of the first line of text, never above it.
