## Key Features Demonstrated

### 1. Authentic CBT Experience
- Split-screen layout matching official IELTS CBT
- Professional, clean design
- Intuitive navigation

### 2. Instant Band Score (7.5)
- Immediate feedback after submission
- Band score: **7.5** (32/40 correct)
- Encouraging personalized messages

### 3. Comprehensive Feedback System
- ✓ Green highlights for correct answers
- ✗ Red highlights for incorrect answers
- Detailed explanations for all questions
- Educational "Key Learning" sections

### 4. Smart Learning Features
- "Show me the section" button
- Yellow highlighting of answer locations
- Visual passage-question linking

### 5. Question Type Coverage
- Multiple Choice (A, B, C, D)
- True/False/Not Given
- All standard IELTS reading question types

### 6. Accessibility Features
- Font size controls (A−, A, A+)
- Timer with visual warnings
- High contrast colors
- Clear visual hierarchy

### 7. Navigation & Controls
- Question navigation buttons
- Timer countdown with color warnings (red at low time)
- Submit Quiz button always accessible
- Clear Highlights feature

1. Navigate to this directory
2. Start a local server: `python3 -m http.server 8889`
3. Open `http://localhost:8889/index.html` in your browser
