# Version 2.21 Testing Guide

## Pre-Testing Setup
1. Update to version 2.21
2. Have a test quiz with at least 40-50 questions for navigation testing
3. Have a CBT quiz with reading passages for highlighting testing

## Test Cases

### 1. Version Update Test
**Objective**: Verify the version has been updated correctly

**Steps**:
1. Navigate to WordPress Admin â†’ Plugins
2. Look for "IELTS Course Manager" in the plugin list
3. Verify version shows as 2.21

**Expected Result**: Plugin version displays as 2.21

---

### 2. Yellow Background Highlighting Test
**Objective**: Verify text highlighting works correctly in CBT reading passages

**Steps**:
1. Open a Computer-Based Test (CBT) quiz with reading passages
2. Select any text in the reading passage on the left
3. Click "Highlight" from the context menu
4. Verify the selected text has a yellow background
5. Refresh the page and verify highlights persist
6. Click "Clear" button to remove highlights

**Expected Result**: 
- Selected text displays with bright yellow (#ffff00) background
- Highlights persist across page refreshes
- Clear button successfully removes all highlights

---

### 3. Question Navigation Sizing Test
**Objective**: Verify all question numbers fit on one line without wrapping

**Steps**:
1. Create or open a CBT quiz with 50+ questions
2. Scroll to the bottom to view the question navigation bar
3. Verify all question numbers and | separators are visible
4. Check that numbers do not wrap to a second line
5. Test on different screen sizes (desktop, laptop)

**Expected Result**: 
- All question numbers display on a single line
- Numbers are smaller (28px wide, 11px font)
- Separators (|) are compact
- If too many questions, horizontal scrolling is available
- Navigation is still readable and clickable

---

### 4. No Answer Selected Feedback Test
**Objective**: Verify custom feedback displays when no answer is selected

**Steps**:
1. Edit an existing quiz question
2. Scroll to "Feedback Messages" section
3. Add text to "No Answer Selected Feedback" field: "Please select an answer before submitting."
4. Save the quiz
5. Take the quiz as a student
6. Submit the quiz WITHOUT answering this question
7. Review the results

**Expected Result**: 
- The custom "No Answer Selected Feedback" message displays for unanswered questions
- If no custom message, falls back to "Incorrect Answer Feedback"

---

### 5. Instructions Field Test
**Objective**: Verify optional instructions display correctly above questions

**Steps**:
1. Edit an existing quiz question (or create new one)
2. Find the "Instructions (Optional)" field at the top
3. Add instructions text, for example:
   ```
   Questions 14-20
   
   Reading Passage 2 has seven sections A-G.
   Choose the most suitable headings for sections A-G from the list of headings below.
   ```
4. Save the quiz
5. View the quiz on the frontend

**Expected Result**: 
- Instructions appear above the question number
- Instructions have blue background with left border
- Text is italicized and clearly distinct from question text
- Instructions support basic HTML formatting

---

### 6. Questions Panel Scrollbar Test
**Objective**: Verify questions panel has scrollbar for long quizzes

**Steps**:
1. Open a CBT quiz with 20+ questions
2. Observe the questions panel (right side)
3. Try scrolling within the questions panel
4. Verify reading text panel (left side) also scrolls independently

**Expected Result**: 
- Questions panel has vertical scrollbar when content exceeds viewport
- Scrolling is smooth and independent from reading text panel
- Both panels scroll independently

---

### 7. Combined Features Test
**Objective**: Test all features working together

**Steps**:
1. Create a new quiz with:
   - Instructions on first question
   - 40+ questions total
   - CBT layout enabled
   - Reading passage with text
   - No Answer feedback set on multiple questions
2. Take the quiz as a student:
   - Highlight some text in the reading passage
   - Navigate through questions using bottom navigation
   - Leave some questions unanswered
   - Submit the quiz
3. Review results

**Expected Result**: 
- All features work harmoniously
- Instructions display properly
- Navigation doesn't wrap
- Highlighting works
- Unanswered questions show appropriate feedback
- No JavaScript errors in console

---

## Known Issues and Limitations
- Navigation buttons become very small with 100+ questions - may require horizontal scrolling
- Instructions field uses basic editor on new questions (full editor available after save)

## Troubleshooting

### Highlighting Not Working
- Clear browser cache
- Check browser console for JavaScript errors
- Verify sessionStorage is enabled in browser

### Navigation Wrapping
- Check screen size - very small screens may still wrap
- Verify CSS changes were applied (hard refresh: Ctrl+F5)

### Instructions Not Saving
- Verify WordPress editor permissions
- Check for PHP errors in debug log
- Ensure wp_kses_post is not stripping content

## Security Testing
- Verify all input is properly sanitized
- Test XSS attempts in feedback and instructions fields
- Verify only authorized users can edit quizzes
