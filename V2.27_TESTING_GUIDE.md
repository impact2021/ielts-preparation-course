# Version 2.27 Testing Guide

## Quick Testing Checklist

Use this guide to verify that version 2.27 is working correctly in your WordPress environment.

## Pre-Testing Setup

1. **Backup your database** before testing (recommended)
2. Ensure you're logged in as an administrator
3. Navigate to WordPress Admin → IELTS Courses → Exercises

## Test 1: Verify All Question Types Are Available

### Steps:
1. Go to **IELTS Courses → Exercises → Add New**
2. Scroll down to **Quiz Settings** meta box
3. Click **Add Question**
4. Click on the **Question Type** dropdown

### Expected Result:
You should see all 13 question types in the dropdown:
- ✅ Multiple Choice
- ✅ Multi Select
- ✅ True/False/Not Given
- ✅ Headings Questions
- ✅ Short Answer Questions
- ✅ Sentence Completion Questions
- ✅ Summary Completion Questions
- ✅ Table Completion Questions
- ✅ Labelling Style Questions

- ✅ Essay

## Common Issues & Solutions

### Issue: Question types dropdown doesn't show new types
**Solution**: Clear your browser cache and refresh the page

### Issue: Multiple choice options field doesn't show for Headings
**Solution**: Make sure you selected "Headings Questions" not "Short Answer Questions"

### Issue: Answer not marked as correct despite being right
**Solution**: Check that correct answer is spelled exactly as expected (though matching is flexible)

### Issue: Can't save exercise
**Solution**: Ensure at least one question is added before publishing
