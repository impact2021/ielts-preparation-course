# IELTS Reading Test - CBT Interface Screenshots

This directory contains demonstration screenshots of the new Computer-Based Testing (CBT) interface for IELTS Reading Tests. These screenshots showcase the modern, professional design that mirrors the official IELTS CBT experience.

## Overview

The new reading test interface provides a split-screen layout similar to the official IELTS Computer-Based Test, with reading passages on the left and questions on the right. This design offers an authentic test-taking experience and includes several advanced features to help students prepare effectively.
