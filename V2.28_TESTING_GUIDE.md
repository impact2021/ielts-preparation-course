# Version 2.28 Testing Guide

## Overview
This guide covers testing for version 2.28, which includes:
1. Plugin version update to 2.28
2. Fix for matching questions display to show individual question numbers

## Test Environment Setup
- WordPress 5.8 or higher
- PHP 7.2 or higher
- IELTS Course Manager plugin installed and activated

## Test Cases

### 2. Matching Questions - Individual Question Numbers (Standard Layout)

**Test ID:** V2.28-002  
**Priority:** Critical  
**Description:** Verify that matching questions display individual question numbers in standard layout

**Prerequisites:**
- Create or use an existing exercise with matching questions
- The matching question should have multiple matches (e.g., 7 matches)

**Steps:**
1. Navigate to IELTS Courses → Exercises
2. Create a new exercise or edit an existing one
3. Add a matching question with at least 7 match items (e.g., questions 7-13)
4. Configure each match with text and correct answer
5. Save and publish the exercise
6. View the exercise on the frontend using the standard layout
7. Observe how the matching questions are displayed

**Expected Results:**
- The question group header should show "Questions 7 – 13" (or appropriate range)
- Each individual match item should display its own question number:
  - First match: "Question 7"
  - Second match: "Question 8"
  - Third match: "Question 9"
  - And so on...
- Each match should have its own answer input field
- Question numbers should be sequential and individual

**Previous Behavior (Bug):**
- All matches were displayed without individual question numbers
- Only the group header showed the question number

**Pass/Fail:** ___________

### 4. Question Numbering Consistency

**Test ID:** V2.28-004  
**Priority:** High  
**Description:** Verify question numbering is consistent across different question types

**Prerequisites:**
- Create an exercise with multiple question types in sequence

**Steps:**
1. Create a new exercise with the following question sequence:
   - Questions 1-3: Multiple choice questions
   - Questions 4-6: True/False questions
   - Questions 7-13: Matching question (with 7 matches)
   - Questions 14-15: Fill in the blank questions
2. Save and publish the exercise
3. View the exercise on the frontend
4. Verify the question numbering is sequential

**Expected Results:**
- Questions 1-3 should be numbered correctly as individual questions
- Questions 4-6 should be numbered correctly as individual questions
- Questions 7-13 should each be numbered individually (not grouped)
- Questions 14-15 should be numbered correctly following question 13
- No gaps or duplicates in question numbering
