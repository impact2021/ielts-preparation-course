# Version 2.12 Testing Guide

This guide provides comprehensive testing instructions for Version 2.12 of the IELTS Course Manager plugin.

## Pre-Testing Setup

### WordPress Environment
- WordPress 5.8 or higher
- PHP 7.2 or higher
- IELTS Course Manager plugin v2.12
- At least one user account with appropriate permissions
- Browser with JavaScript enabled

### Test Data Requirements
1. **Course with Multiple Lessons:**
   - Create a course with at least 3 lessons
   - Each lesson should have at least one exercise

2. **CBT Exercise (with popup enabled):**
   - Computer-Based IELTS Layout enabled
   - "Open as Popup/Fullscreen Modal" checked
   - At least 5 questions
   - Reading text sections added
   - Timer set (e.g., 10 minutes)
   - Band score display mode enabled (optional, test both)

3. **CBT Exercise (without popup):**
   - Computer-Based IELTS Layout enabled
   - "Open as Popup/Fullscreen Modal" unchecked
   - At least 5 questions
   - For comparison with popup mode

4. **Regular Exercise:**
   - Standard quiz layout (not CBT)
   - At least 5 questions
   - For regression testing

## Test 3: CBT Quiz Submission - Basic

**Purpose:** Verify quiz submission works and results modal appears

**Prerequisites:**
- CBT exercise (with popup enabled or disabled)

**Steps:**
1. Open CBT exercise
2. Answer at least 3 questions (mix of correct and incorrect)
3. Leave 2 questions unanswered
4. Click "Submit Quiz" button

**Expected Results:**
- ✅ Button changes to "Submitting..." briefly
- ✅ Results modal appears (for popup mode) OR results show inline (for non-popup)
- ✅ Modal shows:
  - Pass/Fail status
  - Score (e.g., "3 / 5 correct" or "60%")
  - Band score (if band mode enabled)
  - Time limit (if timer was set)
  - Time taken (in MM:SS format)
- ✅ NO detailed question feedback list in modal
- ✅ NO auto-redirect countdown timer

## Test 4: CBT Results Modal - Simplified Display

**Purpose:** Verify CBT results modal does NOT show feedback or auto-redirect

**Prerequisites:**
- CBT exercise with popup enabled

**Steps:**
1. Complete and submit CBT quiz
2. When results modal appears, carefully examine its contents

**Expected Results - Modal SHOULD Show:**
- ✅ "Congratulations! You Passed!" OR "Quiz Completed"
- ✅ Score summary (X / Y correct)
- ✅ Band score or percentage
- ✅ Time limit (if set)
- ✅ Time taken
- ✅ "Review my answers" button (primary blue button)
- ✅ "Take Quiz Again" button
- ✅ "Continue" button (if next lesson/exercise exists)

**Expected Results - Modal SHOULD NOT Show:**
- ❌ Question-by-question feedback section
- ❌ "Question Feedback" heading
- ❌ List of questions with correct/incorrect status
- ❌ Individual question answers
- ❌ Countdown timer (e.g., "Automatically continuing in 5 seconds...")
- ❌ Auto-redirect notice
