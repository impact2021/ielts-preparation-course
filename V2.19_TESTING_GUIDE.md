# Version 2.19 Testing Guide

**Date**: 2025-12-19  
**Version**: 2.19  
**Testing Level**: Manual Testing Required

## Overview

This guide provides comprehensive testing instructions for all features implemented in version 2.19. Follow these test cases to ensure all functionality works correctly before production deployment.

---

## Pre-Testing Setup

### Requirements
- WordPress site with IELTS Course Manager plugin installed
- Access to admin dashboard
- At least one CBT (Computer-Based Test) quiz created
- Test quiz should have:
  - 10+ questions
  - Multiple reading texts
  - Timer enabled (optional, for testing timer integration)
  - Linked to a course (for "Return to course" link)

### Test Environment
- ✅ Desktop browser (Chrome, Firefox, or Safari)
- ✅ Mobile browser (for responsive testing)
- ✅ Different screen sizes (1920x1080, 1366x768, 1024x768)

### Test User Account
- Student account enrolled in test course
- Admin account for verification

### Test 2: Multiple Text Highlights - Basic Functionality

**Objective**: Verify multiple text sections can be highlighted independently

**Steps**:
1. Log in as a student
2. Navigate to a CBT quiz with reading text
3. In the reading column, select first paragraph (or sentence)
4. Right-click and select "Highlight" from context menu
5. Verify yellow background appears on selected text
6. Select a different section of text (separate from first highlight)
7. Right-click and select "Highlight"
8. Verify second section also shows yellow background
9. Add a third highlight in a different location
10. Verify all three highlights are visible simultaneously

**Expected Results**:
- All highlighted sections display yellow background
- Multiple highlights coexist without interfering
- Each highlight is independent

**Pass Criteria**: 
- ✅ Can create multiple separate highlights
- ✅ All highlights show yellow background
- ✅ No limit on number of highlights (tested up to 5+)

**Status**: ⏳ Pending

---

### Test 8: Return to Course Warning - Before Submission

6. Verify page stays on quiz (does not navigate)
7. Verify quiz state is preserved (answers still filled in)
8. Click "Return to course" again
9. Click "OK" in dialog
10. Verify navigation to course page occurs

**Expected Results**:
- Warning dialog appears before navigation
- Cancel keeps user on quiz page
- OK allows navigation to proceed
- Answers are preserved when staying

**Pass Criteria**:
- ✅ Warning dialog appears
- ✅ Cancel prevents navigation
- ✅ OK allows navigation
- ✅ Quiz state preserved

**Status**: ⏳ Pending
